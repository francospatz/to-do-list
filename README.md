# Welcome to to-do-list 👋

> It is made in React with Tailwindcss and Sass for the styles and Framer-Motion for the animations.
> 

### ✨ [Demo](https://francospatz.github.io/to-do-list)

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```
